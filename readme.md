# excel-api

VBAを利用してネット上のAPIにアクセスを行うサンプルです    
サーバー、クライアントの両方を作成しました

## やること
### リスト出力
1. データをCSVファイルから読み込む
2. データをJson形式に変換
3. 出力

### 検索
1. 検索条件を取得
2. データをCSVファイルから読み込む
3. 検索条件に従ってデータを抽出
4. データをJson形式に変換
5. 出力

## そのうちやること
### 検索（DBよりデータ取得）
1. 検索条件を取得
2. 検索条件に従ってデータベースからデータを抽出
3. データをJson形式に変換
4. 出力
